* `git config --global user.name "yourname"`
	* 配置用户名
* `git config --global user.email "youremail"`
	* 配置用户邮箱

* `git init`
	* 初始化git仓库
* `git status`
	* 查看git仓库状态
`git add`
	* 添加文件至暂存区
`git commit`
	* 添加暂存区文件至本地git仓库
`git push`
	* 上传仓库变更内容至远程仓库

`git remote add origin https://www.github.com/yourname/reponame.git`
	* 添加与远程仓库的关联
`git push origin master`                                                                   
	* 第一次上传仓库内容至远程仓库
